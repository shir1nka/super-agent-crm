A full-featured mobile CRM application for car wash businesses. Real-time updates via WebSockets, offline support with local SQLite, and a beautiful UI built with React Native (Expo).

## ✨ Features

- 🔐 **Role-based auth** with biometric fingerprint support
- 📋 **Queue management** — real-time updates via Socket.IO
- 💳 **POS system** — service grid with multiple payment methods
- 📅 **Booking calendar** — schedule and manage appointments
- 📊 **Analytics** — interactive charts with Victory Native
- 📴 **Offline support** — local SQLite + background sync
- 📸 **Camera integration** — vehicle photo capture
- 🔔 **Push notifications** — real-time alerts
- 🌍 **Bilingual** — Russian / Uzbek interface
- 🔒 **Secure storage** — MMKV + Expo Secure Store

## 🛠️ Tech Stack

| Category | Technology |
|----------|-----------|
| **Framework** | React Native, Expo |
| **Language** | TypeScript |
| **State** | Zustand, TanStack React Query |
| **UI** | React Native Paper |
| **Forms** | React Hook Form + Zod |
| **Real-time** | Socket.IO |
| **Charts** | Victory Native |
| **Storage** | MMKV, Expo SQLite, Expo Secure Store |
| **Auth** | Biometric + JWT |

## 🚦 Getting Started

```bash
npm install
npx expo start
```

Scan the QR code with **Expo Go** app on your phone.
