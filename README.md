# Super Agent CRM

Mobile CRM application for car wash businesses. Built with React Native (Expo) with real-time updates and offline support.

## Features

- Role-based authentication (biometric support)
- Queue management with real-time updates via Socket.IO
- POS system with service grid and payment methods
- Booking calendar
- Analytics with interactive charts
- Offline support with local SQLite + sync
- Camera integration for vehicle photos
- Push notifications
- Bilingual interface (Russian / Uzbek)

## Tech Stack

- **Framework:** React Native, Expo
- **Language:** TypeScript
- **State:** Zustand, TanStack React Query
- **UI:** React Native Paper
- **Forms:** React Hook Form + Zod
- **Real-time:** Socket.IO
- **Charts:** Victory Native
- **Storage:** MMKV, Expo SQLite, Expo Secure Store

## Getting Started

```bash
npm install
npx expo start
```

Scan the QR code with Expo Go app on your phone.
