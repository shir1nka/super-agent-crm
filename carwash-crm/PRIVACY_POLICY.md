# Политика конфиденциальности CarWash CRM

**Последнее обновление:** 22 мая 2026 г.

## 1. Какие данные мы собираем

Приложение CarWash CRM собирает следующие данные:
- **Имя и контактные данные:** имя, фамилия, номер телефона, email
- **Данные об автомобилях:** госномер, марка, модель, цвет, VIN
- **История визитов:** даты, услуги, суммы оплат
- **Данные об устройстве:** модель ОС, версия приложения, push-токен
- **Фотографии:** фото автомобилей (до/после мойки), фото чеков
- **Геолокация:** для отображения ближайших филиалов (только с разрешения)

## 2. Как мы используем данные

- Для оказания услуг: запись на мойку, история визитов, оплата
- Для отправки уведомлений: статус мойки, напоминания, акции
- Для анализа: отчёты, загрузка моек, популярность услуг
- Для улучшения сервиса

## 3. Хранение и защита данных

- Данные хранятся на серверах на территории РФ
- Передача данных — по HTTPS
- Пароли хранятся в зашифрованном виде (bcrypt)
- Токены доступа — в защищённом хранилище устройства

## 4. Передача данных третьим лицам

Мы не передаём данные третьим лицам, кроме:
- Платёжные системы (для обработки оплат)
- Облачные уведомления (FCM/APNs для push)

## 5. Права пользователей

- Право на доступ к своим данным
- Право на удаление аккаунта и всех данных
- Право на экспорт данных
